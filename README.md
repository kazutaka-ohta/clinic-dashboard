# 🏥 クリニック経営ダッシュボード

**理学療法士が作った、クリニック向け経営可視化ツール**

CSVと経理Excelをドラッグ&ドロップするだけで、売上・件数・新患数・曜日別収益などを自動集計・グラフ化します。インストール不要。HTMLファイルをブラウザで開くだけで動きます。

---

## 🎯 背景・課題
定期的に行われる院内ミーティングでは、経営データが複数のシステムに分散しているため、現状認識にばらつきが生じやすい状態が続いていました。

月次の売上把握や前年比較のたびに
- 予約システムのCSV（サービス別・PT別の件数）
- 経理システムのExcel（日次の見込み収益）
- 診察システムのCSV（新患・再初診数）
これらを手動でまとめる作業が発生しており、データよりも集計作業に時間がかかっていました。

そこで、理学療法士として現場の業務フローとデータ構造を考慮した上で、これらを統合する経営ダッシュボードを設計・実装しました。売上・収益内訳・サービス件数・新患数・全体トレンドを一画面で確認できるようにし、CSVとExcelをドラッグ&ドロップするだけで自動集計される仕組みにしました。

導入後は、ミーティングでの議論が「データの確認」から「意思決定」に移行し、既存サービスの刷新・新規サービスの導入・集客施策の検討などに活用されています。

---

## ✨ 主な機能

### 📂 ファイル読み込み
- CSV・Excel（.xlsx）をドラッグ&ドロップで読み込み
- 複数ファイルを一括ドロップ対応
- ファイルの種別（予約CSV・診察CSV・経理Excel）を**自動判別**
- 1ファイル複数月（横持ち形式）・1ファイル1ヶ月（縦型）両対応

### 📊 グラフ（10種類）
| グラフ | 内容 |
|--------|------|
| 月次売上（積み上げ棒） | サービス別＋診療部門の内訳 |
| 前年同月比 | 経理Excelがあれば実収入ベースで比較 |
| 初診 / 再初診推移 | DR別の月次新患数 |
| 曜日別見込み収益 | 平均収益を曜日ごとに集計 |
| 月次件数推移 | サービス別の件数推移 |
| PT別件数（累計） | スタッフごとの稼働状況 |
| 月次収入内訳 | 保険・自費・診療部門の構成 |
| 月次売上 + 目標ライン | 目標金額との差額を可視化 |
| 新患・再初診（詳細） | 月別の累計推移 |
| 各DR 月次新患数 | 担当医師別の新患推移 |

### 💰 単価設定
- メニューごとの単価をUI上で設定
- **件数 × 単価**で推計売上を自動計算
- 設定はブラウザに自動保存（localStorage）

### 📋 月別データ一覧
- サービス別・PT別・明細の3タブ切り替え
- 経理Excelがあれば見込み収益・保険・自費も併記

## 🚀 使い方

```
1. clinic_dashboard.html をブラウザで開く
2. ファイルをドラッグ&ドロップ
   - 予約CSV（サービス別件数）
   - 経理Excel（月別シートまたは単一シート）
   - 診察CSV（新患・再初診数）※任意
3. 「ダッシュボードを表示 →」をクリック
4. 年度セレクターで切り替え
```

インストール・サーバー起動・ライブラリのセットアップは**一切不要**です。

## 📁 対応ファイル形式

### 予約CSV
```
サブメニュー, YYYY-MM
予約数, 予約分数, 金額
メニュー名, 件数, 分数, 金額
...
```

### 経理Excel
```
A1セル: 年月（シリアル値）
行5〜35: 日次データ
  A列: 日付（シリアル値）
  C列: 見込み収益
  H列: 保険患者負担
  J列: 自費金額
行36: 月合計
```

### 診察CSV（新患・再初診）
- 縦型（1ファイル1ヶ月）・横持ち（1ファイル複数月）両対応
- 「初診」「再初診」で始まる行を自動認識

## 🛠 技術構成

| 項目 | 内容 |
|------|------|
| 構成 | シングルHTMLファイル |
| グラフ | [Chart.js 4.4.0](https://www.chartjs.org/) |
| Excel読み込み | [SheetJS 0.18.5](https://sheetjs.com/) |
| フォント | Noto Sans JP / DM Mono（Google Fonts） |
| データ保存 | localStorage（単価設定のみ） |
| 外部通信 | なし（全処理がブラウザ内で完結） |

**患者データ・売上データは一切外部に送信されません。**

---

## 📌 設計上の工夫

- **ファイル自動判別**: 1行目の構造とデータ行の内容からファイル種別を判定
- **列名に依存しない**: 年月・件数をヘッダーのパターンマッチで抽出
- **推計 vs 実収入**: CSVの推計売上と経理Excelの実収入を重ねて差分を可視化
- **診療部門の導出**: 見込み収益 − PT推計売上 = 診療部門として自動計算
